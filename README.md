# S.EE for Raycast

Create short URLs, text shares, and file shares with S.EE from Raycast.

## Setup

1. Get an API key from your S.EE account.
2. Open Raycast Settings -> Extensions -> S.EE.
3. Fill in:
   - `API Key` (required)
   - `API Base URL` (optional, default: `https://s.ee/api/v1`)
   - Optional default domains for URL/Text/File commands

## Commands

- Create Short URL
- Create Text Share
- Upload File
- Quick Share
- Manage History
